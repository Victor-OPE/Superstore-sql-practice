# Superstore SQL Practice

This repository contains SQL practice queries written to explore the Superstore dataset.

## Overview
The project focuses on answering key business questions using SQL, including:
- Total sales per region
- Top 5 products by revenue
- Monthly sales trends for 2025
- Average sales per customer
- Total orders by segment

## Skills Practiced
- SQL aggregation and grouping
- Filtering and date formatting
- Query organization and optimization
